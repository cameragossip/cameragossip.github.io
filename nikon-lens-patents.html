Nikon Lens Patents

Introduction

This database attempts to find the best (not necessarily exact) patent match for a production Nikkor lens. Note that the match is not always perfect as the final production lens can sometimes be different from the patent that is identified as the best match. Where possible this is indicated in the table below.

Last Updated 2 July 2022

Additional Resources

It is my hope that one day we will be able to evaluate the optical designs in these patents using freely available Open Source software. As of today the best Open Source software is Michael Hayford's ray-optics. There is a growing collections of patents analysed using ray-optics in this repository.

I also recommend following sites as great sources of information regarding Nikkor lenses.

  • Nikkor Thousand and One Nights
  • Photons to Photos Optical Bench Hub by Bill Claff
  • Red Book Nikkor Aid
  • Rolands Nikon Pages - Nikon Lens Serial Numbers and Manufacturing Dates
  • Marco Cavina's Website
  • Articles by Marco Cavina
  • Pierre Toscani's site

Contributing

If you notice a mistake or have a better patent for a given lens, or want to add more lenses to this list, please send me a pull request at the github source repository for this page. You can also submit an issue.
